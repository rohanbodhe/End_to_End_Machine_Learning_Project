# 🎫 Dynamic Ticket Pricing System

ML-powered price optimization tool achieving **99.1% prediction accuracy**.


## 📊 Project Overview
Predicts optimal ticket prices based on urgency, capacity, weather, competitors, and social buzz.

## 🎯 Outstanding Results
- **Model Accuracy:** 99.1% (R² = 0.991) ⭐
- **Prediction Error:** ±$3.14 RMSE
- **Best Model:** Random Forest (MSE: 9.88)
- **Dataset:** 30,000+ ticket transactions
  
## 🛠️ Tech Stack
- **Python:** Pandas, NumPy, Scikit-learn, XGBoost
- **ML Models:** Random Forest, XGBoost (both 99%+ accurate)
- **Database:** SQLite with 15+ analytical queries
- **Deployment:** Streamlit Cloud
- **Version Control:** Git & GitHub

## 📈 Model Performance
